# Simon-says-skila-verekfni
                        Simon says skýrsla
Það tók föstudags tímana og smá af mánudags tímanum að klára simon says verkefnið.
Það gekk mjög vel að gera það og kláraði á tíma það gekk vel að lóða og setja allt saman.
